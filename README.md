Car Market – React Project

โปรเจกต์เว็บขายรถ ทำด้วย React + Tailwind CSS

วิธีเริ่มใช้งาน
1. ติดตั้ง dependencies
npm install

2. รันโปรเจกต์ (โหมดพัฒนา)
npm start


เปิดที่: http://localhost:3000

3. สร้างไฟล์สำหรับ deploy (โหมดโปรดักชัน)
npm run build

📁 โครงสร้างโปรเจกต์ (ย่อ)
src/
 ├─ components/
 │   └─ Navbar.jsx
 │   └─ imgs/
 │       ├─ logocar.svg
 │       └─ ham-menu.svg
 ├─ App.js
 └─ index.js
